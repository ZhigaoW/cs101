# 内核网络


- Documentation : 此目录下没有内核的代码，有一套有用的内核文档
- arch 不同体系下的特有代码，至少有3个子目录
  1. kernel 不同体系结构内核特有实现方式
  2. lib 不同体系结构下的高新能通用代码的实现
  3. 不同体系结构的特有内存管理程序实现
- drivers : 驱动 -> 包含大部分代码
- fs : 文件系统
- include ： 大部分的(*.h)
- init : 内核的启动
- ipc ： 进程间通信
- kernel : 此目录是linux内核中最重要的，包含内核中平台无关的基本功能
- lib : 通用函数和内核自解压函数
- mm : 平台无关的内存管理代码
- net ： 内核网络协议栈代码
